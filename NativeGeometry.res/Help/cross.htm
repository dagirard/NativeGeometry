Good way to use NativeGeometry


NativeGeometry has been designed to be a cross platform/localizable/dpi aware geometry manager, enabling developpers to develop quickly a good looking application for all platforms.

Use paddings!


NativeGeometry enables you to set distance of objects with the virtual padding rect. To makes your application good looking on all platform simply prefer using (left|top|right|bottom)padding than setting a fixed size in pixel.

Set geometry relations for all objects


When you will localize or change the DPI of your application, all objects have to changes their size.

To prevent object overlapping, simply use NativeGeometry and set strictly the distance between the two objects.

Update automatically fonts of your application


NativeGeometry provides an useful command that enable you to update all fonts of your stack in only one line.
This allow your application to have using the right font on each platform.

The easiest way to update all font is to place inside your mainstacks the following script:

----------------------------------------------
on preopenstack
-- Setting the stack font to the system font.
ng_setStackFont ng_getSystemFont(), the long id of this stack
pass preopenstack
end preopenstack
----------------------------------------------
And do not forget inside substacks to place "pass preopenstack"!

More information about the ng_setStackFont command and ng_getSystemFont() are available inside the NativeGeometry API documentation.

Use scaling instead of setting bigger font!


NativeGeometry 3 added a scaling feature, all objects handled by NativeGeometry can be scaled up/down with a simple command!

So, instead of using the ng_setStackFont command or a manual script to update font size, simply set the general scalling!

Example:
----------------------------------------------
-- Setting the scale value for this platform
ng_setConfig "scale", ng_getSystemScale()
----------------------------------------------

The function ng_getSystemScale() returns the scaling factor to use on this platform.
